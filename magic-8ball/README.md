# Magic 8-Ball in Python

Mini Python project simulating a Magic 8-Ball using random numbers and conditional logic.

## What this project does

- Generates random answers to user questions  
- Uses conditional statements to select responses  
- Displays question and answer in the console  

## Concepts used

- import random  
- randint()  
- if / elif / else  
- variables  
- string concatenation
