# Financial Returns Analysis

Mini Python project calculating weekly and annualized log returns and displaying results as percentages.

## What this project does

- Calculates annualized log return from daily returns  
- Calculates weekly return by summing daily log returns  
- Formats results as percentages  
- Demonstrates that log returns are additive over time  

## Concepts used

- Functions  
- for loops  
- sum()  
- List operations  
- String formatting  
- Financial return calculations
