# Shipping Cost Calculator

Mini Python project calculating shipping costs based on package weight and delivery method.

## What this project does

- Calculates Ground Shipping cost  
- Shows Ground Shipping Premium price  
- Calculates Drone Shipping cost  
- Compares different shipping methods  

## Concepts used

- if / elif / else  
- comparison operators  
- variables  
- basic arithmetic  
- print formatting  

## Shipping methods

- Ground Shipping  
- Ground Shipping Premium  
- Drone Shipping
