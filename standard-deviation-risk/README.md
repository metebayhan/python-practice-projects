# Standard Deviation Risk Analysis

Mini Python project calculating and comparing investment risk using standard deviation.

## What this project does

- Calculates standard deviation using NumPy  
- Calculates standard deviation manually using a custom implementation  
- Compares risk between two assets  
- Displays results as percentages  
- Determines which asset is riskier based on volatility  

## Concepts used

- Standard deviation  
- Variance  
- Mean calculation  
- for loops  
- NumPy  
- Square root (math.sqrt)  
- Financial risk comparison
