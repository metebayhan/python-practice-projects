# Inventory Data Analysis with Pandas

This mini project demonstrates basic Pandas operations:

- Reading CSV files
- Filtering rows with conditions
- Creating boolean columns
- Calculating derived columns
- Using lambda + apply for row-based transformations

## Features

- Identifies in-stock products
- Calculates total inventory value
- Creates combined product descriptions
- Filters seed products in Brooklyn

## Tech Stack

- Python
- Pandas

## How to Run

```bash
pip install pandas
python main.py
