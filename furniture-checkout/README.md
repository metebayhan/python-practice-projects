# Furniture Checkout

Mini Python project simulating a simple customer checkout system.

## What this project does

- Stores product descriptions and prices  
- Calculates customer total  
- Applies sales tax  
- Prints purchased items  
- Displays final amount to pay  

## Concepts used

- Variables  
- String concatenation  
- Basic arithmetic  
- Incremental totals (+=)  
- Simple tax calculation
