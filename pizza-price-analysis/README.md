# Pizza Price Analysis

Mini Python project analyzing pizza toppings and prices.

## What this project does

- Counts toppings with specific prices  
- Displays total number of pizza types  
- Combines toppings and prices into nested lists  
- Sorts pizzas by price  
- Finds cheapest and most expensive pizzas  
- Adds new pizza options  
- Selects the three cheapest pizzas  

## Concepts used

- Lists and nested lists  
- count()  
- sort()  
- pop()  
- slicing  
- indexing
