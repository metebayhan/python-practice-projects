# Gradebook Manager

Mini Python project managing student grades using nested lists.

## What this project does

- Creates a gradebook using subject and grade pairs  
- Adds new subjects and grades  
- Updates existing grades  
- Replaces numeric grades with pass/fail  
- Combines current and last semester gradebooks  

## Concepts used

- Nested lists  
- Indexing  
- append() and remove()  
- List concatenation  
- Variables and basic logic
