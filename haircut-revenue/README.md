# Haircut Revenue Analysis

Mini Python project analyzing haircut prices and weekly revenue.

## What this project does

- Calculates average haircut price  
- Applies a $5 discount to all prices  
- Calculates total weekly revenue  
- Calculates average daily revenue  
- Finds hairstyles under $30 after discount  

## Concepts used

- for loops  
- range() and len()  
- list comprehension  
- list indexing  
- basic arithmetic operations
